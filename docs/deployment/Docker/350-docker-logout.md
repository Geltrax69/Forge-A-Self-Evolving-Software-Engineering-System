# docker logout

***

| Description | Log out from a registry  |
| ----------- | ------------------------ |
| Usage       | `docker logout [SERVER]` |

## [Description](#description)

Log out from a registry. If no server is specified, the default is defined by the daemon.

## [Examples](#examples)

```console
$ docker logout localhost:8080
```
