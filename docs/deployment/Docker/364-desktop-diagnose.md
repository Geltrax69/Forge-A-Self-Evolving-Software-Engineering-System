# docker desktop diagnose

***

| Description | Diagnose Docker Desktop issues      |
| ----------- | ----------------------------------- |
| Usage       | `docker desktop diagnose [OPTIONS]` |

## [Options](#options)

| Option         | Default | Description                |
| -------------- | ------- | -------------------------- |
| `-u, --upload` |         | Uploads the diagnostic ID. |
