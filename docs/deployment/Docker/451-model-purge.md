# docker model purge

***

| Description | Remove all models              |
| ----------- | ------------------------------ |
| Usage       | `docker model purge [OPTIONS]` |

## [Description](#description)

Remove all models

## [Options](#options)

| Option        | Default | Description                  |
| ------------- | ------- | ---------------------------- |
| `-f, --force` |         | Forcefully remove all models |
