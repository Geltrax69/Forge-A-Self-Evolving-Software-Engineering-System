# docker offload stop

***

| Description | Stop a Docker Offload session   |
| ----------- | ------------------------------- |
| Usage       | `docker offload stop [OPTIONS]` |

## [Options](#options)

| Option        | Default | Description                   |
| ------------- | ------- | ----------------------------- |
| `-f, --force` |         | Don't prompt for confirmation |
