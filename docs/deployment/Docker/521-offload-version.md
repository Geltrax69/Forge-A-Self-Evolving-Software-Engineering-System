# docker offload version

***

| Description | Prints the version                 |
| ----------- | ---------------------------------- |
| Usage       | `docker offload version [OPTIONS]` |

## [Options](#options)

| Option    | Default | Description                |
| --------- | ------- | -------------------------- |
| `--json`  |         | Prints the version as JSON |
| `--short` |         | Prints the short version   |
