### [](#how-can-i-write-comments-in-jsx)How can I write comments in JSX?

```
<div>
  {/* Comment goes here */}
  Hello, {name}!
</div>
```

```
<div>
  {/* It also works 
  for multi-line comments. */}
  Hello, {name}! 
</div>
```
